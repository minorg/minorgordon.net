Professional background

I am a freelance software consultant. I mostly work in Java and Python these days, although I've written a lot of C++ in the past. I hold a Ph.D. in Computer Science from the University of Cambridge, where I worked in the Opera Group at the Computer Laboratory. My main areas of expertise are high-performance web servers (the focus of my research as well as subsequent commercial projects), compilers and runtimes for web scripting languages, web application frameworks, and parallel and distributed file systems.

In the past I have contributed to:

  • parallel file systems for cluster supercomputers at NEC High Performance Computing Europe
  • parallel languages for high-performance computing at IBM Research
  • cloud server performance analysis and provisioning at ChartBeat
  • software architecture and back end integrations at Birchbox
